# Geocentric solar system

Geocentric animated representation of the solar system.

Pascal version of this [JavaScript program](https://github.com/Kharoh/Geocentric-Solar-System.git).

## Build

In *Makefile*, correct the path to the [AGGPas][1] units (depending on your installation), then:

```bash
make
```
